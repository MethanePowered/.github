# Methane Powered Graphics

Primary focus of "Methane Powered" organization is the development of open-source and cross-platform framework [Methane Kit](https://github.com/MethanePowered/MethaneKit) to simplify modern 3D graphics development with rendering abstraction API on top of DirectX 12, Vulkan & Metal. Demo application [Methane Asteroids](https://github.com/MethanePowered/MethaneAsteroids) is maintained in a separate repository.
